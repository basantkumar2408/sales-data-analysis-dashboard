# Sales Data Analysis Dashboard

Interactive retail sales dashboard built with Python.

## Features
- 5 charts: Monthly trend, Category pie, Region bar, Units vs Revenue, Discount heatmap
- Filter by Region and Category
- Export dashboard as PNG

## Tech Stack
Python | Pandas | NumPy | Matplotlib | Seaborn | Tkinter

## How to Run
pip install pandas numpy matplotlib seaborn
python sales_dashboard_app.py

## Dataset
5000 synthetic retail transactions | Jan-Dec 2023
